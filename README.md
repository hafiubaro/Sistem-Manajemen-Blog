# Sistem-Manajemen-Blog
Sistem Manajemen Blog (CMS)  yang memungkinkan pengelolaan data penulis, artikel, dan kategori artikel secara  lengkap. Aplikasi dibangun menggunakan PHP, MySQL, dan JavaScript dengan  pendekatan asynchronous menggunakan fetch API sehingga seluruh operasi berlangsung  tanpa reload halaman.
